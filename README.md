# Zimo Site

Astro site for `zimo.no`.

The active direction is Norwegian-first, dark, maritime, technical, and focused on Zimo's underwater camera system.

## Local Development

```sh
npm install
npm run dev
```

Open `http://127.0.0.1:4321`.

## Build

```sh
npm run build
```

The production build is generated in `dist/`.
